Satellite Mandala

2013
リアルタイム処理による映像
LCDプロジェクタ，コンピュータ，スピーカ
サイズ可変

The algorithm takes color data from pixels of a map, which is obtained according to the location of a satellite (web API data to be precise), to finally draw abstract patterns.

人工衛星（のウェブ上のシミュレーション）の位置座標をもとに地表の画像を取得し，そのピクセルの色情報を用いて，絵画が動くように画面が更新され続ける．
